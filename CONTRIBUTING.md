## VSCode

### VSCode

<kbd>Shift</kbd>+<kbd>Ctrl</kbd>+<kbd>P</kbd> &rarr; _Tasks: Run tasks_

| task             | description |
| ---------------- | ----------- |
| `(re)start Live` | Restart Live, or start it if closed.
| `open Log.txt`   | Open Live's `Log.txt` with the system default application for txt files.
